《麻瓜指南》 企划书

企划人：YHB (Github: Discover304)

发布日期：2021/09/14

企划介绍

这是一个定时提供创作点子的，大家一起讨论并进行合作创作，打造IP的长期企划活动的测试。

企划的本质是组建一个计算机科学领域的研究小组。旨在科普电脑和数学领域的知识；提出新鲜的观点看法；回顾人工智能的发展历程，思考未来的研究方向；研究感兴趣的领域并发表论文。

企划背景

作为一个博主，最难过的事情不过两件，一件是题目太多了，写不过来了；另一件是没有题目可写了，感到江郎才尽。每个博主或者创作者都会反复经历这样的困境。

因此，我希望通过团队的力量，实现优势互补，持续输出高质量的原创内容。介于这是我第一次进行这样的企划活动，所以先小规模的进行一次测试企划，积累一点经验，为之后的长期企划铺路。

参加项目的你可以获得

  1. 可以尽情发挥自己的才华。
  2. 可以交到有意思朋友。
  3. 可以为自己的个人研究招募小伙伴。
  4. 可以打破粉丝数0的惨状，开始拥有影响力。
  5. 可能可以成功拥有论文发表经历，提升学术背景。

我们需要什么样的小伙伴

满足一下任意一点即可：

  1. 有充足的时间。
  2. 有类似的创作经历。
  3. 文笔好，希望提升自己的学术背景。
  4. 成绩优异，希望尝试论文等级的写作。
  5. 喜欢团队合作，相信自己能够提出建树性的意见和建议。
  6. 英语能力、编程能力、熟练使用Markdown、数理逻辑能力、企业实习经历为加分项。

测试企划小组人数：10人。

测试企划时间

2021年9月 – 2021年12月

企划期望目标

  1. 成功搭建并运营小组宣传以及资源分享网站。
  2. 小组在知乎发表至少30篇点赞数量高于100的文章。（注明：我的个人博客半年可以增加40篇）
  3. 小组完成至少5篇论文的开题报告。（注明：我曾尝试两个月完成两篇小组论文，开题报告作为前置工作，耗费的时间可以接受。）
  4. 小组至少在Bilibili发布5段播放量超过1w或者点赞数超过100的视频。
